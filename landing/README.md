# VeriAid AI — Landing Page (Phase 1)

> Hackathon MVP v1.0 · Tuan: Frontend Lead · Teman Tuan: Backend + AI + Blockchain
> Generated by Hermes Core, 2026-06-18

## Cara Buka

1. Buka file `index.html` di browser (double-click).
2. Tuan bisa nilai:
   - **UI/UX feel** — clean, trust-forward, modern
   - **Visual hierarchy** — hero → problem → solution → how → tech → CTA
   - **Color & typography** — teal (trust) + gold (kindness), Inter + Plus Jakarta Sans
   - **Interactive** — hover states, smooth scroll, animated pulse dot
   - **Responsive** — mobile (single column) + desktop (2-col hero)

## Design Decisions

| Aspek | Pilihan | Alasan |
|:---|:---|:---|
| Palette | Deep teal (#0F4C5C) + gold (#E8A23B) | Trust (blockchain) + hope (humanitarian) |
| Font | Inter (body) + Plus Jakarta Sans (display) | Modern, Bahasa-friendly, professional |
| Layout | Single page, scroll-based | Standard hackathon landing, max impact in 1 URL |
| Style | Glass-morphism nav + soft gradient blobs | Modern SaaS aesthetic, not "AI hype" |
| Icons | Inline SVG | No external dep, no icon library bloat |
| JS | None | Pure HTML+CSS, faster Tuan review |

## Sections Implemented (sesuai PRD 9.1)

1. **Sticky Nav** — logo + 4 anchor links + 2 CTAs
2. **Hero** — tagline + sub + dual CTA + 3 trust stats + decorative Proof Certificate mock
3. **Problem** — 6 problem cards (icons + title + desc)
4. **Solution** — 3-step cards (Record → Upload → Verify) dengan featured center
5. **How It Works** — vertical timeline 5 steps dengan connecting gradient line
6. **Technology** — dark section, 2 cards (AI + Blockchain)
7. **CTA** — gradient panel dengan dual button
8. **Footer** — 4-column (brand + 3 link groups) + bottom bar

## Design Notes untuk Tuan

- **Hero visual mock** — sengaja gue bikin "Proof of Integrity Certificate" card dekoratif + "Anomaly Detected" stat card untuk nunjukin VISUAL style konten platform. Bisa lo swap dengan mock yang lebih sesuai pas integrasi.
- **Animated pulse dot** di eyebrow — subtle, gak ganggu, nunjukin tech-forward.
- **Smooth scroll** — anchor link nav scroll smooth ke section.
- **Mobile** — di bawah 980px, hero single column. Di bawah 880px, nav links hide (CTA tetep muncul). Tambahin hamburger menu kapan-kapan kalau perlu.
- **A11y basics** — semantic HTML, aria labels, focus states, reduced-motion support.
